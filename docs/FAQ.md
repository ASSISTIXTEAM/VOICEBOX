# FAQ

**Нужно ли интернет-соединение?**
Для установки зависимостей и загрузки модели Whisper требуется интернет. После загрузки всё работает офлайн.

**Можно ли использовать GPU?**
Да, если у вас установлен PyTorch с поддержкой CUDA. Укажите `--device cuda` или выберите GPU в интерфейсе.

**Где сохраняются результаты?**
По умолчанию рядом с исходным файлом. Формат можно выбрать: TXT, JSON, SRT, VTT или TSV.

**Как ускорить обработку?**
Используйте меньшую модель (`tiny` или `base`) или подготовьте аудио заранее (моно, 16 kHz).
