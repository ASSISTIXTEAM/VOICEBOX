# Сборка VOICEBOX

## Windows
1. Запустите `BUILD_FINAL.bat`.
2. Дождитесь завершения PyInstaller.
3. Готовый результат: `dist/VOICEBOX/VOICEBOX.exe`.

## Linux / macOS
1. Убедитесь, что установлены зависимости (`pip install -r requirements.txt`).
2. Запустите `./build.sh`.
3. Готовый результат появится в `dist/VOICEBOX/`.
